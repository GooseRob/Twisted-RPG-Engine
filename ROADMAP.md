# Twisted RPG Engine - Roadmap

This roadmap outlines planned features, improvements, and long-term goals for
the Twisted RPG Engine. Items are grouped by priority and scope.

---

# Short-Term Goals (0–3 months)

- Finalize battle engine refactor
- Add more status effect types
- Improve AI decision-making
- Expand event runner actions
- Add admin panel improvements
- Add documentation for all major systems
- Improve error handling and logging

---

# Mid-Term Goals (3–9 months)

- Party-based battles
- Multi-target skills and AoE support
- Buff/debuff stacking rules
- Improved elemental system
- Quest chain editor in AdminSauce
- NPC schedule system (day/night cycles)
- Map triggers and region-based events
- Player housing prototype

---

# Long-Term Goals (9–24 months)

- Full MMO-style overworld
- Instanced dungeons
- PvP arenas and ranked ladders
- Crafting and gathering systems
- Mounts and movement upgrades
- Player-run shops
- Server clustering and sharding
- Plugin/modding API

---

# Stretch Goals (Future)

- Mobile client
- VR/AR experimental client
- AI-driven dynamic quests
- Procedural map generation
- Cloud-hosted “Engine-as-a-Service” platform

---

# Philosophy

The roadmap is flexible and community-driven.  
Features are prioritized based on:

- Creator needs
- Engine stability
- Long-term maintainability
- Player experience
